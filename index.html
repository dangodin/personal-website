Projects

Some of the projects I have been working on.

RPi Media Center

Raspberry Pi (RPi) can be an inexpensive, fun little toy for geeks. Embedded with a low cost, low power ARM processor it has use as a web server, a media center, or an Internet of Things (IoT) node or mote. My personal RPi currently runs the OpenElec media center software.

The RPi is quick to set up, requiring only basic Linux skills. The result: remarkably smooth streaming of high definition video from a networked hard drive.

Bootstrap Website

The geniuses at Twitter released their Bootstrap framework as open source in 2011 and it is a great way to spin up consistent clean looking website front-ends relatively quickly. This very website is designed using Bootstrap, and it did not take long.

You can find Bootstrap powering many popular web apps and it appears to be an excellent platform for this purpose.

Cloud Hosting

In the heated struggle between my website and media center for Raspberry Pi domination, the media center eventually won out. Thus, the shunned but resilient website needed some good ol' fashion cloud hosting to really shine.

There are ever increasing options for low cost cloud hosting and they are easier than ever to spin up. I chose DigitalOcean primarily based on price. After installing Apache, writing a couple quick backup scripts, and migrating the site over it is now running beautifully.

Google Analytics

Many companies use Google Analytics to gather detailed statistics that help them measure, understand, and optimize website usage behaviour. However, one thing that can be said about using it for your personal website, it's probably a clear indication that you are a data junky.

Deploying this service is as simple as signing up for a free (basic) account and pasting a snippet of JavaScript code onto each webpage that you wish to to monitor.
